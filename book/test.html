Welcome to Top Python Libraries

Top Python Libraries is a Medium community for writers focused on the Python programming language.

Anyone who loves writing is welcome to join, regardless of experience.

Please read the following guidelines to ensure your submission is not ignored or rejected.

Accepted Topics

Any Python-related topics are welcome, such as:

  • Python Programming Basics.
  • Python Data Science.
  • Python Machine Learning.
  • Python Data Visualization.
  • Python Automation Scripts.
  • Python Web Development.

Any Python-related topic not listed above is also accepted.

Style Guidelines

  • Titles and Subtitles: Properly format titles and subtitles using Medium tools. Try adding a kicker above the title.
  • Images: Use images from Pexels, Unsplash, or other open-source sites. All images must have titles. For more information on adding image titles, check this guide.
  • Section Breaks: Add section breaks where needed.
  • Headers and Sub-Headers: Use headers and sub-headers where needed.
  • Introduction and Conclusion: Include an introduction and a conclusion (you don’t have to label them specifically).

Please note the following rules:

  • Fully AI-generated articles will not be accepted.
  • Submit unpublished drafts only! Any published articles will be rejected.
  • If you need help with Medium formatting tools, refer to this link.
  • Use Grammarly (free) to avoid typos and misspellings.

How to Become a Writer

We are looking for writers passionate about Python who want to share their knowledge and insights with our readers. Whether you are a researcher, developer, content creator, or simply fascinated by this technology, we want to hear from you.

To become a writer, just leave your Medium username in the comment section.

We’d appreciate it if you clap for this article and follow the publication.

Submission

Follow this tutorial to submit your story and learn how to add a draft or post to the publication.

If you don’t hear back from us within two days, please assume we have politely declined your submission.

That’s it. We look forward to reading your submissions and sharing your insights with our readers.

Together, let’s explore the possibilities of Python!
